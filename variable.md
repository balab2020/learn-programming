# How to define variable

## JavaScript 

```js
var a;                          // variable
var b = "hello";                // string
var c = "Hi" + " " + "Bala";    // = "Hi Bala"
var d = 1 + 2 + "3";            // = "33"
var e = [2,3,5,8];              // array
var f = false;                  // boolean
var g = /()/;                   // RegEx
var h = function(){};           // function object
const PI = 3.14;                // constant
var a = 1, b = 2, c = a + b;    // one line
let z = 'zzz';                  // block scope local variable
```
