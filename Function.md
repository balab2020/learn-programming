# How to define function

> Sum of 2 Numbers

Function which accepts two numbers as arguments and return the sum.
```
Function(A, B)
START
     RETURN (A+B)
END
```
## JavaScript 

```js
function sum(a,b)
{
   return (a+b);
}

//or

const sum = (a,b) => a+b;
```

## C#

```csharp
class Calculator 
{
    public int Sum(int a, int b)
    {
	return (a+b);
    }
}
```

## Python

```python
def sum(a,b):
    return a+b
```
