# Print *Hello World* string in Console

## JavaScript 

```js
var helloWorld = "Hello World";
console.log(helloWorld);
```

## C#

```csharp
class ConsoleApplication 
{
    static void Main(string[] args)
    {
	    var helloWorld = "Hello World";
		Console.WriteLine(helloWorld);
    }
}
```

## Python

```python
helloWorld = "Hello World";
print(helloWorld);
```
